# Setup

Setup scripts.

## Ubuntu

Development script
```sh
sudo su -c 'wget -O develop.sh https://raw.githubusercontent.com/lidj22/setups/main/ubuntu-22/develop.sh \
    && chmod +x develop.sh \
    && ./develop.sh \
    && rm develop.sh'
```
