# Setup

Setup scripts.

## Ubuntu

Development script
```sh
sudo su
wget -O ubuntu.sh https://raw.githubusercontent.com/lidj22/setups/main/ubuntu-22/develop.sh \
    && chmod +x ubuntu.sh \
    && ./ubuntu.sh \
    && rm ubuntu.sh
```
